Component wrapper for [ClapprPlayer](https://github.com/clappr/clappr).

This component uses some plugins rebuild and hosted in npm. For more details check the plugins docs.

## Installing

```bash
npm install --save v-clappr-player

# or

yarn add v-clappr-player
```

### Import globally in app

```javascript
import VClapprPlayer from 'v-clappr-player';

Vue.use(VClapprPlayer);
```

### Import into your component
```js
import VClapprPlayer from 'v-clappr-player'

// add to component
export default {
  components: {
    VClapprPlayer,
  }
}
```

### Installing on Nuxt

```js
// plugins/v-clappr-player.js
import Vue from 'vue'
import VClapprPlayer from 'v-clappr-player'

Vue.use(VClapprPlayer)
```

```js
// nuxt.config.js
export default {
  ...
  plugins: [
    { src: '~/plugins/v-clappr-player', ssr: false }
  ],
  ...
}
```
