# Usage

After instaling the component use it in your application.

```js
<template>
  <v-clappr-player
    id="example"
    source="https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4"
    width="100%"
    :options="options"
    :aspect-ratio="16 / 9"
  />
</template>

<script>
import VClapprPlayer from 'v-clappr-player'

export default {
  components: {
    VClapprPlayer
  }
}
</script>

```

## Accessing the player

You can access clappr player to use any of it's methods. Check a list of methods at [clappr docs](https://github.com/clappr/clappr-core#books-api-documentation).

```js
<template>
  <v-clappr-player
    ref="clappr"
    id="accessing-player"
    source="https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4"
    width="100%"
    :aspect-ratio="16 / 9"
  />
</template>

<script>
import VClapprPlayer from 'v-clappr-player'

export default {
  components: {
    VClapprPlayer
  },
  methods: {
    playVideo () {
      this.$refs.clappr.player.play()
    }
  }
}
</script>
```
