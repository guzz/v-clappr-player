# Theming

### Controls
You can configure the `seekbar` and `buttons` colors by passing an `mediacontrol` property in the options.

### Poster
Pass a `poster` property to define an image to be shown before the video is played.

```js
<template>
  <v-clappr-player
    ref="clappr"
    id="themed-player"
    source="https://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4"
    width="100%"
    :options="options"
    :aspect-ratio="16 / 9"
  />
</template>

<script>
import VClapprPlayer from 'v-clappr-player'

export default {
  components: {
    VClapprPlayer
  },
  computed: {
    options () {
      return {
        mediacontrol: { seekbar: '#f39d52', buttons: 'green' },
        poster: 'https://i.pinimg.com/originals/df/51/0f/df510f0dd46630f457df9f88d3379691.jpg'
      }
    }
  }
}
</script>
```
